Validate(bool force)

Validates the current object for field and business rule errors.

Parameters

Name Description
force If set totrue, a validation is forced. When the validation is not forced, it means that when the object is already validated, and no properties have been changed, no validation actually occurs since there is no reason for any values to have changed.

Remarks

To check whether this object contains any errors, use the ValidationContext property.
