    public abstract class CommandBehaviorBase<T> : BehaviorBase<T> where T : FrameworkElement

    Base types BehaviorBase

    Behavior base class that handles a safe unsubscribe and clean up because the default behavior class does not always callOnDetaching. This class extends the Missing: <see cref=“T:Catel.Windows.Interactivity.BehaviorBase`1” /> class by adding supports for commands.

    Type Parameters

    T The this behavior should attach to.
